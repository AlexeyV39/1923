Как загрузить:
1. Открой github.com
2. Создай новый репозиторий, например: card-games-pro
3. Нажми Add file → Upload files
4. Распакуй этот ZIP и перетащи ВСЕ файлы в репозиторий
5. Нажми Commit changes
6. Открой Settings → Pages
7. Source: Deploy from a branch
8. Branch: main / root
9. Save
10. Жди 2–10 минут
